# Data Structure and Sorting Algorithms

These are small projects that are implementations of some data structures and sorting algorithms. These were my first steps in programming and were what really developed my love it. All these small programs are written in C++. They are for educational purposes. The data structure files how a simple way of implementing a binary_search_tree, dequeque, hashtable, linked_list and vector. The sorting algorithm files include simple functions that are restricted to either either strings or ints.
